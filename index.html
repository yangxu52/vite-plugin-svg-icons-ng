vite-plugin-svg-icons-ngA high-performance SVG icon plugin for Vite

Options Guide
🚀

Pre-generation & Injection

Automatically generates SVG sprites from local files and injects them at runtime, enabling global reuse with the same output.

⚡

High Performance

Combines async build processing with caching, updating icons only when needed for a faster, smoother experience.

📦

Out-of-the-box

Sensible defaults and comprehensive docs for quick setup and a unified icon system.

🔥

HMR Support

Icon changes are pushed through a custom sprite update event, so the existing sprite DOM is replaced in place without a full page reload.

🧩

SSR Ready

Works seamlessly in SSR environments, ensuring consistent rendering without extra setup.

🧱

Consistent Output

Ensures identical SVG sprite output across dev, build, and SSR to avoid environment inconsistencies.
